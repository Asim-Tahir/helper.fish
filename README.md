## Install

[Oh My Fish][omf]:

```sh
omf install https://github.com/Asim-Tahir/helper.fish
```

[Fisher][fisher]:

```sh
fisher install Asim-Tahir/helper.fish
```

## Functions

After installing the [`helper.fish`][repo] plugin, you can use available functions:

### `fish_remove_path`

Remove specified path from the `$PATH` environment variable.

```sh
fish_remove_path /path/to/remove
```
